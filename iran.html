Latest recommendations for Iran

From a thread on r/dumbclub:

  • Use a clean IP address (not blocked due to a previous user)
  • Use xtls-rprx-vision to prevent TLS-in-TLS discovery
  • Use Reality to eliminate TLS fingerprint of the server
  • Use uTLS to prevent client TLS fingerprint detection
  • Use your own SSL certificate ("steal_oneself") instead of borrowing from a foreign website
  • Use decoy fallback to evade SNI detection
  • Block domestic IP addresses and domains to prevent correlation detection (chocolate4u)
